Scribbit's Tech Stack

Scribbit is a robust and feature-rich digital scribe for your notes.

More details

  • Google Cloud API Computer Vision and NLP (with AutoML) Scribbit is powered by Google's Cloud APIs and in-house Machine Learning. The photo to text and handwriting processing is handled by the Computer Vision API, while the entity classification and topic modelling are handled by the Natural Language Processing API.
  • Flask Backend Framework Scribbit is built on the lightning-fast Python-based Flask framework. The note syntax was inspired by the intuitive design of markup languages.
  • Bulma Frontend/CSS Framework Scribbit's UI/UX are styled with Bulma. The design philosophy was inspired by the vision of a seamless and modular workspace.
  • Muuri & Quill.JS JS Libraries Scribbit's workspace modularity is in part powered by Muuri, and its editing capabilities by Quill.js.
